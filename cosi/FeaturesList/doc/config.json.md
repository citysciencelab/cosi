**FeaturesList**

Eine Übersicht der Features die auf der Karte zu sehen sind.

|Name|Verpflichtend|Typ|Default|Beschreibung|
|----|-------------|---|-------|------------|
|name|nein|String|Einrichtungsübersicht|Name des Werkzeuges im Menu.|
|icon|nein|String|bi-apple|CSS Klasse des Glyphicons, das vor dem Toolnamen im Menu angezeigt wird.|

**Beispiel**
```
"featuresList": {
    "name": "Einrichtungsübersicht",
    "icon": "bi-apple"
}
```
